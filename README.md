### **Descripcion General**
Este repositorio contiene la coleccion completa de ejercicios desarrollados durante el curso de PHP Puro. El objetivo principal fue dominar las bases del lenguaje, el manejo de datos y la implementacion de practicas de seguridad en el desarrollo web.

### **Contenido del Curso**

#### **Modulo 1: Fundamentos de Programacion**
*   **02_hola_mundo.php**: Primer contacto con etiquetas PHP e impresion de texto.
*   **04_variables.php**: Manejo de tipos de datos, nomenclatura camelCase y asignacion.
*   **06_operadores.php**: Implementacion de operaciones aritmeticas y logicas.
*   **08_condicionales.php**: Control de flujo mediante estructuras IF y validaciones con funciones isset y empty.
*   **12_bucles.php**: Automatizacion de tareas mediante ciclos While, For y Foreach.

#### **Modulo 2: Estructuras de Datos y Logica**
*   **19_funciones.php**: Desarrollo de funciones modulares con tipado estricto (Type Hinting).
*   **22_arreglos.php**: Manipulacion de arreglos indexados y tecnicas de depuracion.
*   **23_arreglos_asociativos.php**: Gestion de datos mediante claves personalizadas.
*   **28_modularizacion.php**: Organizacion de archivos mediante las sentencias include y require.

#### **Modulo 3: Web y Persistencia**
*   **31_get.php / 31_recibir.php**: Comunicacion de datos a traves de la arquitectura cliente-servidor mediante GET.
*   **32_post.php / 32_recibir.php**: Procesamiento seguro de formularios mediante el metodo POST.
*   **35_archivos.php**: Sistema de carga y gestion de archivos en el servidor.
*   **40_sesiones.php / 40_perfil.php**: Control de sesiones de usuario y persistencia de navegacion.

#### **Modulo 4: Arquitectura Avanzada**
*   **45_poo.php**: Fundamentos de Programacion Orientada a Objetos (Clases, Objetos y Constructores).
*   **50_pdo_conexion.php**: Establecimiento de conexiones seguras a bases de datos MariaDB/MySQL.
*   **51_pdo_consultas.php**: Utilizacion de PHP Data Objects (PDO) y Sentencias Preparadas para prevenir ataques de Inyeccion SQL.
